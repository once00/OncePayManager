# OncePayManager


## 安装
[OncePayManager使用方法](http://onceo.me/)
#### CocoaPods
在 Podfile 中添加 pod 'OncePayManager'。
执行 pod install 或 pod update。
导入 <OncePayManager.h>。

## 手动安装
1.下载 OncePayManager 文件夹内的所有内容。

2.将 OncePayManager 内的OncePayManager源文件添加(拖放)到你的工程。

## 系统要求
该项目最低支持 iOS 8.0 和 Xcode 8.0。
